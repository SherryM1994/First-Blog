由零开始创建个人博客2（安装Git）

第一次安装Git，要吐槽一下，国内网直接在官网下载Git真的很困难，网速那叫一个慢啊，后来才知道要下载Git应该用外网，有点麻烦，在万能的乎友帮助下，找到了一个下载捷径，废话就不说了，直接上链接

腾讯软件中心 这个是腾讯软件中心，关键时刻还是腾讯爸爸给力，进到界面选择对应的版本就可以了，因为我的是Windows系统，所以接下来的操作都是从Windows角度出发

当然你也可以从官网下载git

接下来就是安装过程

1、点击next

2、选择安装目录，我一般不会把软件装在C盘，我选择安装在D盘

3、在默认选项的基础上勾选Additional icons

4、Select Start Menu Folder，继续保持默认选项，next

5、选择编辑器，继续保持默认选项，next

6、选择路径环境

7、继续保持默认选项，next

8、继续保持默认选项，next

9、继续保持默认选项，next

10、继续保持默认选项，next

开始安装

11、finish

12、桌面空白处右键，点击Git Bash Here

弹出窗口

输入git命令

如果出现这样的结果就是安装成功了
