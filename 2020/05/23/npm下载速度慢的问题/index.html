用npm install遇到的坑

内网用npm命令来安装express、Vue、webpack真的很慢

遇到同样问题的小伙伴可以考虑通过淘宝镜像来安装npm，这样速度会快一些

我才有的方法是将淘宝镜像地址加到配置文件中，这样以后每次下载都会使用这个淘宝服务器来下载，不需要再重新输入了

下载之后可以通过npm config list来查看npm的配置信息

之后就可以直接用npm install命令来下载配置了
