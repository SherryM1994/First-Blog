由零开始创建个人博客1.1（注册GitHub）

作为一个零编程基础的菜鸡，想用知乎记录一下自己学习编程所踩过的坑，也方便以后回过头看看曾经为了过不一样人生的自己有多努力，做人对得起自己不愧于心便够了。写的不好，各位大佬要是看到的话千万不要吐槽，要脸（捂脸跑）。

至于为什么要做个人博客呢，一开始就是为想做点东西，让自己有点成就感吧，想想能用来记录自己入坑的轨迹和生活琐事，感觉也不错，说干就干，虽然学了点蹩脚的html、css还没学js，很多东西都没学，边学边做吧，这样也许会快一些，然后再查漏补缺。

因为这个blog我打算用git+hexo写的，所以第一步是注册GitHub账号

什么是GitHub，GitHub是一个面向开源及私有软件项目的托管平台。GitHub于2008年4月10日正式上线，除了git代码仓库托管及基本的 Web管理界面以外，还提供了订阅、讨论组、文本渲染、在线文件编辑器、协作图谱（报表）、代码片段分享（Gist）等功能。目前，托管版本数量非常之多，其中不乏知名开源项目 Ruby on Rails、jQuery、python 等。

登录GitHub官网GitHub

1、点击sign up进入页面进行注册

2、进行注册

ps:这里的邮箱QQ邮箱也是可以的

3、填完信息，点击NEXT:Select a plan 进入下一步Choose a plan，选择账户类型

GitHub 的仓库分为两种，一种是public repositories公开免费版，一种是private repositories私有付费版。其中，免费版是完全公开的，而私有版一般是由企业或者不愿公开仓库的个人用户购买，因为只是为了搭建个人博客和日常使用，选择免费版的就可以了，点击Choose Free，之后Creat a account

4、接下来会跳转到一个调查问卷，直接skip the step跳过就可以了

5、接下来会进入邮箱验证环节

打开邮箱点击verify email address

6、之后会跳转到登录界面，填写完毕后sign in

7、之后邮箱会收到一个验证码

将验证码填写到里面就可以登录账号了

人生充满着未知，有想法就要去做，反正今日不知明日事，只求来日不后悔。继续探索人生道路，同时感谢在我生命中帮助过我的人。

​ 一只在苦逼生活中努力挣扎的咸鱼
