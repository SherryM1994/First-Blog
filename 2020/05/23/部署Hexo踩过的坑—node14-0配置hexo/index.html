部署Hexo踩过的坑—node14-0配置hexo

hexo -v出现的问题

hexo d -g 出现的问题

出现这些是因为node版本太高，切换成低版本的node来安装Hexo就可以了

我原先是安装了最新版node14.0，后来多装了一个比较稳定的node12.14版本，这个问题就解决了
